# Sistema de automatização de contratos - Exercício sobre Interfaces em java

Projeto desenvolvido como exercício do curso de Java do professor Nélio Alves, focando nos conceitos de interface entre classes, upcasting e boas práticas de orientação a objetos.

## Funcionalidades

-  Cadastro de contratos, com data do contrato, valor e numero de parcelas para pagamento mensal.
  
## Tecnologias Utilizadas

   - Java SE

   - Git/GitHub

   - Markdown (documentação)

##  Como Executar

### Pré-requisitos

- Java JDK 25 ou superior
- Git (para clonar o repositório)

### Passo a Passo
Pode utilizar uma IDE como Intellij ou VSCode

ou

1. **Clone o repositório:**
```bash
git clone https://github.com/MarcosJuniorDev/ExercicioComposicaoJava.git
cd ExercicioComposicaoJava
```
2. **Compile o projeto e execute a aplicação**
```bash
javac -d bin src/entities/*.java src/enums/*.java src/*.java
java -cp bin Main
```
